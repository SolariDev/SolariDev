## 🚀 Proyectos actuales

🔵 **THBR** – Proyecto institucional en PHP/WordPress  
Proyecto en producción, con repositorio privado.  
La aplicación web permite cargar contratos de alquiler de propiedades inmuebles, facilita el seguimiento de vencimientos, tiene una interface que muestra el nivel de urgencia semaforizado y se comunica automáticamente por mail con las partes involucradas cuando es necesario.   

🟢 **Rentify** – Evolución de THBR  
El plugin en desarrollo activo de WordPress, está mejorado como plantilla, con su propio Child-Theme a partir de Hello-Elementor.  
Busca ofrecer una experiencia más clara y escalable para la gestión de contratos.  
Es la base de mi trabajo actual y seguirá creciendo con nuevas funciones.
